Bind Mount - Managed by User , Stored in host path, less secure
volume - Managed by docker, stored in docker area, safer

docker run --mounttype=<type>, source=<src>, target=<dest> image

type- volume, bind, tmpfs
source- volume name or host path
target- path inside container
readonly- mount as readonly
